## v0.2.14

### Bug Fixes

- **Clear focus border on terminal reset** — When a single pane's shell exits, the blue focus border is now explicitly removed along with the cursor and scrollback.
- **Extend vertical box-drawing lines to bottom edge** — Vertical box-drawing characters (│, ┤, etc.) in the last row now stretch to the view's bottom edge, matching the right-edge treatment from v0.2.13.
- **Respawn terminal at current dimensions** — Re-activating an exited terminal now preserves the current rows/cols instead of resetting to hardcoded 24×80.
