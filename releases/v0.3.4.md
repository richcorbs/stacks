## Bug Fixes

- **Fixed memory leak in wide character rendering** — `NSString` and `NSMutableAttributedString` objects created in `drawWideChars` (for emoji, CJK, and symbol characters) were never released, leaking on every frame. Added `autorelease` so they're cleaned up by the per-frame autorelease pool. This caused steady ~3 MB/min growth under active terminal usage.

- **Suppress quit confirmation during update** — Clicking "Update Now" in the update dialog no longer shows the "Quit Stacks?" confirmation, since the user already confirmed by choosing to update.
