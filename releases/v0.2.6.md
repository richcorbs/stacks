## Improvements

- **Sidebar UI polish**: Selected terminal now displays as a rounded pill shape with horizontal margin, matching modern macOS design patterns.
- **Nav highlight matches pill**: Keyboard navigation (⌘⇧[/]) shows a rounded blue border pill instead of a full-width box.
- **Cleaner project layout**: Removed background highlighting from project headers. Added spacing before project dividers.
- **Readable ANSI colors**: Brightened the 16-color ANSI palette for dark backgrounds, especially blue which was nearly invisible.

## Bug Fixes

- **Color palette sync**: Fixed indexed color fallback table not matching the vterm palette, causing color changes to have no effect.
