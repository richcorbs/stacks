## Improvements

- **⌘⇧[] sidebar navigation fix** — the shortcut sometimes required two presses because the navigation cycle included "+ Add Terminal" buttons between terminals. Now it only cycles through actual terminals.
- **Removed "+ Add Terminal" sidebar button** — redundant with ⌘T and the new context menu option. Cleans up the sidebar layout.
- **"Add Terminal" in project context menu** — right-click a project header to add a terminal to that specific project.
- **Sidebar spacing** — improved bottom padding below each project's terminal list.
