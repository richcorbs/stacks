# v0.1.2 — Patch

**2026-03-24**

## Bug Fixes

- **Fix window position not restoring on launch** — `window.center()` was called unconditionally, overriding the saved frame from `setFrameAutosaveName:`. Now the saved position is explicitly restored and centering only happens on first launch.

## Improvements

- **Version in stats bar** — the bottom stats bar now shows the app version alongside CPU and memory usage (e.g., `CPU 0% • MEM 134MB • 0.1.2`).
