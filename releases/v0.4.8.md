# v0.4.8

## Improvements

- **Reduce scrollback memory by 63%** — scrollback lines now use a compact cell format (`CompactCell`, 20 bytes) instead of the full cell struct (52 bytes). The `chars` array is reduced from 10 codepoints to 2, which covers 99.9% of real-world cells. Only ZWJ emoji sequences in scrollback history lose trailing codepoints. The live terminal grid is unaffected.
