## v0.2.13

### Bug Fixes

- **Eliminate right-edge and bottom-edge gaps in terminal rendering** — The last column and last row background fills now extend to the view edges, closing the visible strip caused by the grid not being an exact multiple of cell dimensions.
- **Extend box-drawing lines to view edge** — Horizontal box-drawing characters (─, ┬, etc.) in the last column now stretch to the right edge of the view, preventing broken lines in TUI applications.
- **Cursor kept at normal size** — The cursor block uses standard cell dimensions even at edges, so it doesn't appear stretched.
