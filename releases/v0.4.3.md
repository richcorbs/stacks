## Features

- **⌘1–⌘9 quick jump** — instantly switch to any terminal by position. ⌘9 always jumps to the last terminal. Holding ⌘ shows shortcut badges in the sidebar.

## Bug Fixes

- **Fixed copy adding extra spaces around emojis** — wide characters (emojis) had their continuation cells copied as spaces. Now correctly skips libvterm's `0xFFFFFFFF` continuation markers.
- **Copy preserves combining characters** — encodes all codepoints per cell, handling flag emojis, skin tone modifiers, and ZWJ sequences.

## Improvements

- **Cleaned up sidebar** — removed the PROJECTS header bar and "+" button (redundant with ⌘O). Adds vertical space for the project list.
- **Top border on sidebar** — matches the main content area for visual consistency.
- **"Add Terminal" in project context menu** — right-click a project header to add a terminal.
