## Stacks v0.4.13

### Features

- **Maximize the focused pane inside a terminal** — added a temporary maximize toggle for the active pane in split terminal layouts.
- **Terminal-only shortcut** — use **⇧⌘Enter** while a terminal pane is focused to maximize or restore that pane without affecting sidebar navigation.

### Improvements

- **Maximized pane highlight** — maximized panes now use a lime green focus border so the temporary zoomed state is visually distinct.
- **Shortcuts menu discoverability** — added **Shortcuts → Terminal actions → Maximize Pane** to document the maximize shortcut.
- **Terminal-scoped action routing** — maximize is handled by the terminal responder instead of the app delegate, so the shortcut only works when a terminal has focus.

### Bug Fixes

- **ObjC class registration safety** — bumped the custom AppKit class names after changing methods, avoiding duplicate-class registration issues during development and relaunches.
