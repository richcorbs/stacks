## Stacks v0.4.14

### Bug Fixes

- **Fix terminals becoming unopenable after heavy create/delete churn** — deleting running terminals could leave their in-memory sessions behind, eventually exhausting the 32-session limit so newly selected terminals would no longer open until Stacks was restarted.
- **Fix terminal deletion cleanup order** — Stacks now destroys the live terminal session before removing the sidebar/store entry, avoiding use-after-free of the terminal ID during session teardown.

### Documentation

- **Refresh README feature list** — updated the README to reflect recent additions including pane maximize, collapsible projects, inline images, unfocused pane dimming, and the About dialog.
