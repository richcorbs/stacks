## Stacks v0.4.11

### Features

- **Collapsible projects in the sidebar** — click a project name to expand or collapse its terminal list.
- **Persisted collapsed state** — sidebar project expansion state is now saved and restored across launches.

### Improvements

- **Cleaner project headers** — removed the disclosure glyph from project rows for a simpler look.
- **Refined project row spacing** — tightened excess padding below project names and adjusted vertical placement for a more balanced sidebar layout.
- **Collapsed-aware sidebar behavior** — sidebar layout and keyboard navigation now respect hidden terminal rows inside collapsed projects.
