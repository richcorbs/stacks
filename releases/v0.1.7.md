## Bug Fixes

- **Split pane ratios persist**: Dragging a divider to resize split panes now saves the ratio, so your custom pane sizes survive quit/reopen.

## Internal Improvements

- **Module extraction**: Refactored `term_text_view.zig` into 6 focused modules with 35 unit tests:
  - `split_tree.zig` — Split pane tree operations and serialization (12 tests)
  - `scrollback.zig` — Ring buffer for terminal history (5 tests)
  - `selection.zig` — Text selection state and helpers (7 tests)
  - `terminal_state.zig` — Terminal/session type definitions (5 tests)
  - `term_keys.zig` — Key code to escape sequence mapping (5 tests)
  - `box_drawing.zig` — Unicode box drawing character lookup (6 tests)

- **Updated documentation**: Architecture docs now reflect the new module structure.
