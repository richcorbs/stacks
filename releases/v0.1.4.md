## Bug Fixes

- **Preserve split ratios when closing panes** — Previously, closing a pane rebalanced all split ratios, resetting any custom sizing. Now the remaining panes keep their adjusted sizes.

- **Fix "command not found" for terminals with startup commands** — Terminals configured with a startup command were launched with `/bin/zsh -l -c`, which only sources `.zprofile`. Changed to `/bin/zsh -lic` so `.zshrc` is also sourced, giving commands access to the full PATH (Homebrew, nvm, etc.).
