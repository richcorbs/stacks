## Stacks v0.4.12

### Improvements

- **Prevent collapsing the active project** — clicking a project name no longer collapses that project when the focused terminal belongs to it.
- **Sidebar feedback toast** — when collapse is blocked, Stacks now shows a toast message explaining why.
- **Wider toast layout** — increased toast width so longer messages are fully visible.

### Polish

- **Reusable toast helper** — generalized the existing copied-to-clipboard toast implementation so it can be reused for other UI feedback.
