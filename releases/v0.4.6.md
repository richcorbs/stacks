# v0.4.6

## Features

- **Kitty graphics protocol for inline images** — Terminal apps that use the Kitty graphics protocol (like pi) can now display images inline. Supports:
  - Transmit+display (`a=T`) with PNG format
  - Chunked base64 transfers across multiple APC sequences
  - Image delete commands (`a=d`)
  - Cell size query response (`CSI 16t`) so apps calculate correct image dimensions
  - Cursor advancement past placed images per Kitty spec
  - Image overlay rendering on the terminal cell grid via CoreGraphics
  - Image cleanup on terminal clear (⌘K) and deduplication on redraws

## Environment

- Set `KITTY_WINDOW_ID`, `TERM_PROGRAM=stacks`, and `COLORTERM=truecolor` in PTY environment so TUI apps can detect graphics and true color support.
