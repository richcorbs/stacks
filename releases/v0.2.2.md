## Bug Fixes

- **Speech-to-text no longer triggers accidentally**: Fixed false activation when typing capitals (Shift+key), using keyboard shortcuts (⌘T, ⌘D), or when dialogs steal focus.
  - Any keypress now cancels pending/active speech immediately
  - First-responder check ensures speech only activates in focused terminal
  - Focus loss (dialogs, pane switches) cancels speech via `resignFirstResponder`
  - Exact modifier comparison prevents activation with Shift+other modifiers
