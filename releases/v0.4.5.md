# v0.4.5

## Bug Fixes

- **Respect cursor visibility escape sequences** — TUI apps (like pi, htop, etc.) that hide the cursor via `\e[?25l` no longer show a stray white block cursor. The terminal now tracks libvterm's `VTERM_PROP_CURSORVISIBLE` state and only draws the cursor when visible.

## Improvements

- **Updated app icon** — new mint gradient design with four shades from bright to muted, replacing the previous monochrome white tiles.
