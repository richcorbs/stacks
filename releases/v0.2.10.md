# v0.2.10

## Features

- **Edit Project dialog** — right-click a project in the sidebar and select "Edit" to change its name or directory. Name is displayed in the sidebar; changing the directory affects only future terminals.
- **Add Project dialog** — ⌘O now shows a dialog with Name and Directory fields (with a "..." browse button) instead of jumping straight to a folder picker. If no name is given, it defaults to the directory basename.

## Improvements

- **Dialog layout refresh** — labels now appear above input fields with dimmer secondary text and more vertical spacing between field groups. Applied to all dialogs (Add/Edit Project, Add/Edit Terminal).

## Bug Fixes

- **Fixed emoji/wide character positioning** — emoji and CJK characters now render in the correct grid cells using the same Y-flip coordinate transform as the main text renderer.
- **Fixed right-side padding gap** — reduced excessive width deduction in terminal size calculation and added full-view background fill to eliminate dark strips at pane edges.
- **Fixed memory leak in project updates** — old name/path strings are now freed before replacement.
