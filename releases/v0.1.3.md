# v0.1.3 — Patch

**2026-03-24**

## Bug Fixes

- **Window position preserved after auto-update** — the updater now explicitly saves the window frame and flushes UserDefaults to disk before relaunching, so the window restores to its previous position and size after an update.

## Improvements

- **Release Notes button in update dialog** — the update prompt now has three buttons: "Update Now", "Release Notes" (opens the GitHub release page in your browser), and "Later".
