# v0.1.0 — Initial Release

**2026-03-24**

First public release of Stacks — a native macOS terminal emulator built in Zig.

## Features

- **Multiple projects** with named terminals per project
- **Split panes** — horizontal (⌘D) and vertical (⇧⌘D) splits with draggable dividers
- **Sidebar** with drag-and-drop reordering of terminals and projects
- **Full terminal emulation** via libvterm (xterm-256color)
- **Scrollback** — 10,000 line history with mouse wheel scrolling
- **Git branch display** in header bar with live cwd tracking
- **Keyboard navigation** — ⌘⇧[] to highlight sidebar items, ⌘Enter to activate
- **Font size controls** — ⌘+/⌘-/⌘0
- **Copy/paste** — select to copy, ⌘V to paste with bracketed paste mode
- **File drag-and-drop** into terminals (auto-escapes paths)
- **Bell notifications** — blue dot in sidebar for background terminal activity
- **Process status indicators** for command-based terminals
- **Persistent state** — projects, terminals, splits, cwds, and window frame saved across launches
- **Auto-update checker** — checks GitHub releases on startup and hourly
- **Custom box-drawing** — Unicode box characters rendered with CoreGraphics for pixel-perfect table borders

## Architecture

- Pure Zig 0.15 with ObjC runtime bindings for AppKit
- libvterm for VT100/xterm emulation
- PTY/fork for shell processes
- CoreText for text rendering
- ~6,000 lines of code
